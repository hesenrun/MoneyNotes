# MoneyNotes
Android记账小程序，java做的
实现的功能：
  1. 简单的记账
  2. 查询功能（今日，本周，本月）
  3. 简单图表
  4. 帐户/分类自定义
  5. 指纹锁支持（防君子不防小人）
  6. 祖安模式（嘴臭的老哥喜欢）
